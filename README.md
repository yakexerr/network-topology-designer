# Программный комплекс "Network Topology Designer"

## 1. Введение

Данный программный комплекс представляет собой десктопное приложение для полуавтоматического проектирования топологий сетей передачи данных. Приложение разработано на языке Python с использованием библиотеки PyQt6 и позволяет пользователю последовательно пройти четыре основных этапа проектирования сети: от построения отказоустойчивой топологии до финальной оценки ее стоимостных и временных характеристик.

## 2. Установка и зависимости

Для работы приложения необходим интерпретатор Python 3.10+ и следующие библиотеки:

- **PyQt6**: Основная библиотека для создания графического интерфейса.
- **pyqt6-tools**: Инструменты для работы с UI-файлами (Qt Designer).
- **openpyxl**: Библиотека для чтения данных из файлов формата `.xlsx`.

**Установка зависимостей:**
Для быстрой установки всех необходимых библиотек можно использовать менеджер пакетов `pip` и файл `requirements.txt`:

```bash
pip install -r requirements.txt
